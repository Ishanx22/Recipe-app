# AI Recipe Finder App

A full-stack web application that suggests recipes based on ingredients entered by the user. Powered by the DeepSeek Inference API for intelligent, AI-driven recipe generation.

## 🔧 Features
- Add ingredients dynamically like a notes app
- Generate recipe suggestions using AI
- Clean and responsive UI built with React

## 🛠️ Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js (optional if using custom server)
- AI Integration: DeepSeek Inference API
